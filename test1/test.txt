
  в JavaScript ключевое слово this обозначает объект, которому принадлежит данный код. значением this, когда оно используется в объекте, является сам объект.
  глобальный объект это функции и переменные доступные в любом месте программы. к свойствам глобального объекта можно обращаться напрямую. если переменная объявлена с помощью var ,то она становится свойством глобального объекта. 
  при неявном связывании функция будет ссылаться на объект и возвращать его значение 
  явное связывание используется если нужно вызвать функцию используя какой-то объект. для этого используются call blind и apply.  call и apply первым аргументом принимают this. call принимает аргументы списком через запятую, apply принимает аргументы в виде массива. 
bind не вызывает функцию сразу, он возвращает другую функцию — связанную с указанным контекстом,  конекст не изменяется. 
  новая привязка создается с помощью оператора new. он позволяет создать экземпляр объекта заданого пользователем типа или одного из встроенных типов. оператор new имеет больший приоритет по сравнению с явным и неявным связыванием
